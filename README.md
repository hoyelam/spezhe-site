# Spetra Website

Static marketing site for Spetra, designed for GitHub Pages.

## Structure
- `index.html`: Marketing landing page
- `privacy.html`: Privacy policy
- `terms.html`: Terms of use
- `assets/`: Styles and imagery

## Update checklist
- Replace `assets/demo-placeholder.svg` with a real demo image or video
- Update the "Last updated" dates in `privacy.html` and `terms.html`
- Confirm contact email and policy language

## Deploy
Push this folder as its own repository and enable GitHub Pages on the main branch.
